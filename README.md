# StruckOut

## Usage

Example if using Servez.
```bash
$ sudo npm -g install servez
$ git clone git@github.com:satoyuichi/StruckOut.git
$ cd StruckOut
$ servez
```
Then in your browser go to http://localhost:8080/.
